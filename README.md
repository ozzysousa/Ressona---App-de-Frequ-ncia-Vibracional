# Ressona---App-de-Frequ-ncia-Vibracional
O Ressona é um aplicativo criado para auxiliar usuários no alinhamento de sua frequência vibracional e na manifestação de intenções. A aplicação combina a definição textual de uma intenção, a gravação de ressonância de áudio (para amplificação da frequência) e a visualização opcional de imagens, tudo gerenciado em um histórico pessoal.
